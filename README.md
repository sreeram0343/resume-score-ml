# Resume Score Checker

A production-grade Resume Score Checker powered by ML.

## Features
- **Backend**: FastAPI, PostgreSQL, Redis, XGBoost
- **Frontend**: Next.js 14
- **Deployment**: Docker Compose

## Quickstart

1. Copy `.env.example` to `.env` and configure.
2. `make install` to install local dependencies.
3. `make dev` to start the Docker Compose cluster.
